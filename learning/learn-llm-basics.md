# 大语言模型基础学习

## 学习资料

[36k stars 大语言模型入门课程](https://github.com/mlabonne/llm-course)

课程分为三个部分：LLM 基础、LLM 科学家和 LLM 工程师，涵盖了神经网络、Python、自然语言处理（NLP）等基础知识，并提供了关于模型微调和量化的 colab 笔记本。系统且全面，比较基础，适合入门学习。
